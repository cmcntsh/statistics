Assignments

  • Install R and RStudio on your computer.
  • Complete the week 1 RStudio assignment.
    • Download the Week1.zip file from the link at the bottom of the page.
    • Put the zip file in the folder where you want to keep your work for this course.
    • Extract the files.
    • Open RStudio.
    • Open the project file by going to File - Open File... and navigating to the extracted folder and selecting the Week 1.Rproj file.
    • Click on the Files tab in the bottom right pane in RStudio if it's not already showing.
    • You should see the files in the extracted folder.
    • Click on the Week1.Rmd file.
    • It should open in the top left pane in RStudio.
    • Follow the instructions in the document to complete the assignment.
    • You will learn how to import an Excel file, a CSV file, and an SPSS data (.sav) file into RStudio.
  • Complete the week 1 statistics assignment.
    • Practice order of operations: https://www.khanacademy.org/math/cc-sixth-grade-math/cc-6th-arithmetic-operations/cc-6th-order-of-operations/e/evaluating-numerical-expressions-with-exponents
    • Practice summation notation: https://www.khanacademy.org/math/ap-calculus-ab/ab-integration-new/ab-6-3/e/sigma-notation
