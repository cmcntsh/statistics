Assignments

  • Complete the week 3 RStudio assignment.
    • Download the Week3.zip file from the link at the bottom of the page.
    • Put the zip file in the folder where you want to keep your work for this course.
    • Extract the files.
    • Open RStudio.
    • Open the project file by going to File - Open File... and navigating to the extracted folder and selecting the Week 2.Rproj file.
    • Click on the Files tab in the bottom right pane in RStudio if it's not already showing.
    • You should see the files in the extracted folder.
    • Click on the Week3.Rmd file.
    • It should open in the top left pane in RStudio.
    • Follow the instructions in the document to complete the assignment.
    • You will learn how to explore a data set using RStudio.
