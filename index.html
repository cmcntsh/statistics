Proposed Syllabus

Statistics for Healtcare Informatics

Some resources for learning applied statistics using R

Overview

These materials are intended to teach the following process for analyzing data in an applied setting: opening a dataset, data exploration and cleaning, running descriptive statistics, running desired inferential statistical tests, and interpreting and reporting results. I intend to answer three questions about each step of this process: what, why, and how. What should be done at each step? Why should it be done? How can it be done? Since these materials are intended to focus on statistical analyses and using R, I will only touch lightly on topics like study design.

Topics

  • Relationship between informatics and statistics
  • Introduction to statistics and R software
  • Descriptive statistics
    • Organizaing and graphing data
    • Describing distributions
  • Inferential statistics
    • Probability
    • Sampling
    • t-tests
    • Non-parametric tests
    • ANOVA
    • Correlation
    • Linear regression
    • Multiple regression
    • Bayesian approaches
  • Statistics for quality: control charts, run charts, etc.

Activities

  • Readings
  • Lectures (videos)
  • Discussions
  • Assignments
  • Quizzes
  • Tests

Text

  • Field, A. (2013) Discovering statistics using IBM SPSS statistics 4th edition. Sage Publications Ltd. London, Thousand Oaks, New Delhi, Singapore.
